# ClashOfCoins agentic claimer

Скрипт делает три шага:

1. Создает EVM-кошелек (если файла с ключом еще нет) и сохраняет приватный ключ в `wallet1.txt`.
2. Открывает страницу `https://clashofcoins.com/agentic?ref=H-bxcl-ohgv-7048`.
3. Подставляет EVM-провайдер в страницу, проходит wallet-login и нажимает кнопку claim.

## Установка

```bash
npm install
cp .env.example .env
```

## Запуск

```bash
npm start
```

## Важные переменные

- `PRIVATE_KEY_FILE` — путь к файлу с приватным ключом (по умолчанию `wallet1.txt`).
- `RPC_URL` — RPC нода EVM-сети.
- `HEADLESS` — `false` для видимого браузера, `true` для headless режима.
- `CLAIM_TIMEOUT_MS` — таймаут ожидания кнопки и загрузки.

## Ограничения

- На некоторых dApp может использоваться кастомный wallet-connector, который проверяет конкретные расширения браузера. Тогда может потребоваться доработка селекторов и поведения провайдера.
- Для успешного клейма кошелек должен иметь нужную сеть и газ.
